
# Miniblog

Simple miniblog


## Tech Stack

**Client:** React, TailwindCSS( [daisyui](https://daisyui.com/) )

**Server:** Laravel + Inertia + Spatie


## Installation

Clone Project

```bash
  git clone https://github.com/attnmhd/simple-blog.git
```

Install packages

```bash
  composer Install
  npm install
```
